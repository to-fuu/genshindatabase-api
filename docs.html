API Reference

Characters

Returns a detailed list of all characters in the game

Parameters

Name: Returns only the character with said name

Element: Returns characters of said element

Rarity: Returns characters of said rarity

Request: Get all characters
 genshindb-api.herokuapp.com/characters
Response
Request: Get characters by name
 genshindb-api.herokuapp.com/characters/keqing
Response
